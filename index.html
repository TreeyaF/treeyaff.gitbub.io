请配置Firebase以启用协同合作功能：

📚 Firebase设置指南

  1. 访问 Firebase Console
  2. 创建新项目或选择现有项目
  3. 在项目设置中找到"您的应用"部分
  4. 点击"Web应用"图标添加应用
  5. 复制配置信息到上面的输入框
  6. 在"Realtime Database"中创建数据库
  7. 设置数据库规则为测试模式（临时）

数据库规则示例：

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

注意：生产环境请设置适当的安全规则
